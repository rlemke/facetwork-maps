About this data

California cities & routes by zoom
Populated places from OpenStreetMap, tiered by population and served as zoom-tiled PMTiles vector tiles: only the tiles intersecting the current viewport are fetched, and smaller places fade in as you zoom in. Layers: routes 5M, routes 1M, routes 500K, routes 10K, cities 5M, cities 1M, cities 500K, cities 10K. Built by an FFL workflow on Facetwork (fwh_osm).
